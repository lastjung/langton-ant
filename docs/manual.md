# 랭턴의 개미 시뮬레이션 운영 매뉴얼 (Standard Manual)

본 문서는 시뮬레이션의 논리 구조, 색상 매핑 및 작동 규칙에 대한 **최종 표준**을 정의합니다. 모든 개발과 운영은 이 기준을 엄격히 준수합니다.

---

## 1. 상태 및 색상 정의 (State & Color Mapping)

시뮬레이션의 모든 상태는 숫자와 색상으로만 정의하며, "바탕"이나 "흔적" 등의 용어는 사용하지 않습니다.

| 숫자  | 실제 색상 (RGB)      | 논리적 역할             | 규칙 대응 (Classic RL) |
| :---- | :------------------- | :---------------------- | :--------------------- |
| **0** | **검정색 (#0B0E14)** | **시작 상태 (Initial)** | **R** (오른쪽 회전)    |
| **1** | **녹색 (#32D74B)**   | **반전 상태 (Flipped)** | **L** (왼쪽 회전)      |

---

## 2. 표준 작동 로직 (Standard Logic: RL)

개미는 현재 서 있는 칸의 숫자에 따라 기계적으로 행동합니다.

1.  **개미가 0번(검정색)에 서 있을 때:**
    - 현재 방향에서 **오른쪽(R)**으로 90도 회전합니다.
    - 현재 칸의 숫자를 **1번(녹색)**으로 바꿉니다.
    - 앞으로 한 칸 전진합니다.

2.  **개미가 1번(녹색)에 서 있을 때:**
    - 현재 방향에서 **왼쪽(L)**으로 90도 회전합니다.
    - 현재 칸의 숫자를 다시 **0번(검정색)**으로 되돌립니다.
    - 앞으로 한 칸 전진합니다.

---

## 3. 시뮬레이션 환경 표준 (Environment)

- **초기화:** 모든 보드판은 시뮬레이션 시작 시 **0번(검정색)**으로 가득 차 있습니다.
- **개미 비주얼:** 개미는 현재 방향을 가리키는 **핑크색(#FF375F) 화살표**로 표시됩니다.
- **개미 시작:** 정중앙에서 시작하며 초기 방향은 **위쪽(UP)**입니다. 첫 번째 이동은 규칙에 따라 **오른쪽(Right)**이 됩니다.
- **경계 처리 (Edge Behavior):** 개미가 화면 끝(Edge)에 닿으면 반대편에서 나타나는 **워프(Wrap-around)** 방식을 사용합니다. (토러스 구조)
- **학습 모드 (Discovery):** 교육적 목적을 위해 속도가 시뮬레이션 모드 대비 **0.1배**로 저속 Throttling 됩니다.

---

## 4. 규칙 프리셋 가이드 (Presets Guide)

시뮬레이션에서 선택 가능한 주요 패턴의 특징은 다음과 같습니다.

| 프리셋 이름            | 규칙 (Rules) | 패턴 특징                                                                |
| :--------------------- | :----------- | :----------------------------------------------------------------------- |
| **Classic (Standard)** | `RL`         | 10,000보의 카오스 이후 **필연적인 고속도로**를 생성합니다.               |
| **Square**             | `RLR`        | 카오스 단계가 매우 짧고, **사각형의 대칭형** 확장을 보여줍니다.          |
| **Symmetrical**        | `LRRRRRLLR`  | 마치 눈결정이나 복잡한 **기하학적 대칭** 문양을 형성합니다.              |
| **Tower**              | `LLRR`       | 중앙을 중심으로 **수직 또는 수평으로 쌓아 올리는** 듯한 패턴을 만듭니다. |
| **Chaos Edge**         | `RLLR`       | 패턴의 경계가 매우 불규칙하며, **예측 불가능한 확산**을 보여줍니다.      |

---

## 5. 요약

- **0 ➔ 1:** 검정이 녹색으로 변함 (개미가 길을 만듦)
- **1 ➔ 0:** 녹색이 다시 검정으로 변함 (개미가 길을 지움)

이 규정은 수학적 오리지널 정석을 따르며, 어떠한 경우에도 임의로 변경하지 않습니다.
